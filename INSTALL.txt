Stunts Control Installation


--- config.xml
1. If first install, rename config.default.xml to config.xml
2. Edit your config.xml like the description in the file.


--- control.sh
1. Open control.sh
2. Set "cd /home/StuntersControl" to the path where your StuntersControl data is located


--- Upload
1. Upload all files to your server
   

--- chmod
1. Set chmod 777 to your StuntsControl Folder
2. Set chmod 777 to /var/run/StuntsControl.pid


Start StuntsControl with
./control.sh start
